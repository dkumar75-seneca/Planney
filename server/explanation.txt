This file lists the server modules that are planned to implement the scheduling functionality together:

helpers.js: Has all the functions commonly used by more one of the other modules.

accountValidator.js: Handles all of the user identity validation procedures (including getting user access level).

accountManagement.js: Handles all of user account management procedures (signup, access rights and password reset).

requestValidator.js: Handles all of the user input data validation procedures.

requestFormatter.js: Formats the user request into a database connector compatible query.
